《Web应用基础》课程结业报告

作者：范梅浩

指导老师：王勇



一、项目背景

    为了更好的运用所学的web知识来为当前的大学生创新创业项目增光添彩，开发此官网界面。 目的是为了让期颐养老app用户通过网页的形式简单快速地了解我们项目的内容。（当然，最最重要的当然是想WEB结课实践拿高分啦！）


二、项目涉及知识

2.1  CSS+HTML+JAVASCRIPT

    CSS,HTML,JS为本课程重点内容，在本项目中得到了重点体现。CSS负责UI设置，HTML负责具体元素陈列，JS负责网页的逻辑跳转。

    本项目结合了CSS、HTML、JS，呈现了较为美观的界面，达到了简单的跳转界面要求。

2.2  BootStrap框架

    Bootstrap是美国Twitter公司的设计师Mark Otto和Jacob Tho rnton合作基于HTML、CSS、JavaScript 开发的简洁、直观、强悍的前端开发框架，使得 Web 开发更加快捷。Bootstrap提供了优雅的HTML和CSS规范，它即是由动态CSS语言Less写成。Bootstrap一经推出后颇受欢迎，一直是GitHub上的热门开源项目，包括NASA的MSNBC（微软全国广播公司）的Breaking N ews都使用了该项目。国内一些移动开发者较为熟悉的框架，如WeX5前端开源框架等，也是基于Bootstrap源码进行性能优化而来。

     王勇老师在教学过程中推荐了以下框架来完成结课作业BootStrap、MDB、tailwind。再考虑熟练度和美观角度，我选择了BootStrap框架。搭建了如主页所示的界面：

四、遇到的问题

在探索动态网站时，本来想用Bmob后端云构建云服务器，但是由于不太熟悉，最后还是转为静态网站开发。




五、如何解决

我认为，一个人的能力是有限的，只有处于一个分工明确的团队当中，才能发挥巨大的作用。如果仅仅是项目开发而言，若我不能很好的掌握 动态网站的知识，但是我的某个组员可以很好胜任此工作，我们团队仍能取得不错的成就。但是对于学习而言，还是得继续钻研，通过各种帮助文档和博客来解决动态网站的搭建。




六、未解决

如上文所说，为解决的问题有动态界面，之后完善我们的创新创业项目时会联系其他组员，完成该方面的工作。




七、未解决

已依据我十分喜欢的话总结我的学习过程，并且也是对未来自己的督促吧：脚踏实地，才能翱翔远方。
